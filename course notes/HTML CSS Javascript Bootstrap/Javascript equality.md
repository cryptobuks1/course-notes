# Javascript equality

const a = true
    
    a == true //true
    a === true //true
    
    1 == 1 //true
    1 == '1' //true
    
    
    

## Strict equality

    1 === 1 //true
    1 === '1' //false

## Two objects are never equal to each other

## NaN

    NaN == NaN   // false always!

## Null and undefined

    null == undefined  // true
    
    null === undefined // false
